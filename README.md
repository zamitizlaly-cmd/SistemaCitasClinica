#Sistema de Administración de Citas Clinicas

## Instalación y Configuración
1. Clonar el repositorio.
2. Abrir el proyecto en IntelliJ IDEA.
3. Asegurarse de tener configurado JDK 11 en el IDE (File > Project Structure > SDK).

## Uso del Programa
Ejecutar la clase principal "Main.java" desde IntelliJ. El sistema solicitara credenciales de administrador para acceder al menu de gestion de doctores y pacientes.
